# 行为准则 / Code of Conduct

TunnelRiskStudio 欢迎土木工程、BIM、岩土、软件和数据可视化社区的贡献。  
TunnelRiskStudio welcomes contributions from civil engineering, BIM, geotechnical, software, and data visualization communities.

## 期望行为 / Expected Behavior

- 保持尊重和建设性。 / Be respectful and constructive.
- 清楚说明工程假设。 / Explain engineering assumptions clearly.
- 使用可复现示例。 / Use reproducible examples.
- 避免发布敏感项目数据。 / Avoid publishing sensitive project data.

## 不可接受行为 / Unacceptable Behavior

- 骚扰或人身攻击。 / Harassment or personal attacks.
- 未经许可分享私有工程文件。 / Sharing private engineering documents without permission.
- 将原型输出描述为认证工程结论。 / Presenting prototype outputs as certified engineering conclusions.

## 执行 / Enforcement

维护者可以编辑、隐藏或关闭违反这些期望的讨论。  
Maintainers may edit, hide, or close discussions that violate these expectations.
