    创新点 / Innovation

    本项目以开源土木工具的技术路线为启发，重新设计为轻量化浏览器研判台，核心包括 风险因子归一化、场景化权重、工程阈值说明、 可解释输出、BIM/监测数据接口预留。 Inspired by open-source civil engineering tools, this project redesigns the workflow as a lightweight browser-based studio with normalized risk factors, scenario-based weights, engineering threshold notes, explainable outputs, and reserved BIM/monitoring data interfaces.

    监测融合 / Monitoring Fusion

    把沉降、收敛、水位、支护滞后、建筑物距离和掌子面压力统一映射到 0-100 风险域。
    Map settlement, convergence, groundwater, support lag, building distance, and face pressure into a 0-100 risk domain.

    可解释评分 / Explainable Scoring

    输出每个风险因子的贡献值，便于工程师追溯风险来源，而不是只看一个黑箱分数。
    Show each factor's contribution so engineers can trace why the score changes instead of reading a black-box number.

    工程化扩展 / Engineering Extension

    预留 IFC/BIM、传感器 CSV、CPT/钻孔资料、有限元结果导入方向，README 中给出路线图。
    Reserve paths for IFC/BIM, sensor CSV, CPT/borehole records, and finite-element results, with a roadmap in the README.
